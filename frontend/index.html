Find exact duplicates and videos that are segments of longer videos — even with different filenames or re-encodings.

🎞️
Frame Sampling
Each video is sampled at 1 frame/sec, creating a manageable snapshot sequence.
🔢
pHash Fingerprint
Each frame gets a 64-bit perceptual hash via DCT. Similar frames → similar hashes (low Hamming distance).
🔴
Duplicate Detection
Videos with similar lengths are compared hash-by-hash. Low avg distance = duplicate, even if re-encoded.
🟡
Partial Match (Sliding Window)
Short video's hash array is slid across the longer video's array to find the best-matching segment.
